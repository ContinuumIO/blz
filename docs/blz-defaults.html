Defaults for BLZ operation¶

You can tailor the behaviour of BLZ by changing the values of certain some special top level variables whose defaults are listed here. You can change these values in two ways:

  • In your program: the changes will be temporary.
  • In the defaults.py module of the BLZ package: the changes will be persistent.

List of default values¶

eval_out_flavor¶

The flavor for the output object in eval(). It can be ‘blz’ or ‘numpy’. Default is ‘blz’.

eval_vm¶

The flavor for the output object in eval(). It can be ‘blz’ or ‘numpy’. Default is ‘numexpr’, if installed. If not, then the default is ‘python’.
